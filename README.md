# Grocery Guru 🛒💸

Tired of overspending on groceries? **Grocery Guru** is here to help you become a savvy shopper! This application takes your grocery list and scours nearby stores to find the best deals, ensuring you save money on every trip.

## The Problem We Solve 🤔

Grocery shopping can be time-consuming and expensive. Comparing prices across different stores is a hassle, and it's easy to miss out on significant savings. Grocery Guru automates this process, making it simple to find the lowest prices for the items you need.

## Key Features ✨

*   **Smart List Analysis:** Input your grocery list, and Grocery Guru gets to work.
*   **Nearby Store Search:** Automatically identifies grocery stores in your vicinity.
*   **Deal Finder:** Compares prices for your items across all identified stores.
*   **Optimal Store Recommendation:** Tells you which store offers the best overall value for your entire list.

## User Stories 🧑‍🤝‍🧑

*   As a user, I want to upload my grocery list and know which store will give me the best deal so that I can save money effortlessly.
*   As a user, I want to see how much I am saving by using the app so that I can understand the value it provides.
*   As a user, I want to see the itemized costs at all nearby stores for my list so that I can make an informed decision about where to shop, even if it's not the absolute cheapest overall.

## Technology Stack 💻

*   **Backend:** Java
*   **Frontend:** React, Node.js
*   **Database:** MongoDB
